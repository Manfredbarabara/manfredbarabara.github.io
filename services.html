I am responsible for behavior and visuals that run in the user browser so that the users can view and interact with a particular website they are visiting. With the continuing development for mobile devices, such as smart phones and tablets, I usually ensure that my sites come up correctly in browsers on all devices. As a developer, I chiefly concerned with render time, by manipulating the HTML, CSS and JavaScript to ensure that the site opens up quickly.
